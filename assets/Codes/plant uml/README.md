# PlantUML Use Case Diagrams

This folder contains the PlantUML versions of the simplified actor use-case diagrams from `assets/Usecase Diagram/`.

The diagrams intentionally show user goals only. Internal API steps, database tables, and controller method names are not represented as top-level use cases.

Files:

- `usecase-overview.puml`
- `usecase-visitor.puml`
- `usecase-buyer.puml`
- `usecase-owner.puml`
- `usecase-agent.puml`
- `usecase-company.puml`
- `usecase-admin.puml`
